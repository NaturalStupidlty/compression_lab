# Compression algorithms
## Multimedia lab 3

### Installation (Linux)
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Usage
1. Add `./src` folder to the PYTHONPATH

```bash
export PYTHONPATH=$PYTHONPATH:$(pwd)/src
```

2. Run the main script
```bash
python3 main.py
```
